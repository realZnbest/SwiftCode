# GitHub Push Checklist

Use this before pushing or submitting the project.

## Repository

- [ ] Confirm `Trasher.swiftpm` opens in Swift Playgrounds.
- [ ] Confirm `Trasher.swiftpm` builds in Xcode.
- [ ] Confirm no local cache files are staged, such as `.build/`, `.DS_Store`, or `xcuserdata/`.
- [ ] Confirm `README.md` explains that `Trasher.swiftpm` is the main project.

## Gameplay

- [ ] Play once with normal motion settings.
- [ ] Play once with Reduce Motion enabled.
- [ ] Test the street dodge gesture.
- [ ] Test the canal fork choice.
- [ ] Test the recycling bin sorting gesture.
- [ ] Confirm the game reaches the ending without user input.

## Submission

- [ ] Zip only the final `.swiftpm` package when submitting to the contest.
- [ ] Check the zipped file is below 25 MB.
- [ ] Prepare a short project description explaining the plastic bottle journey and recycling message.
